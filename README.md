# fc-trails
Mobile web application for walking trails in Fulham Cemetery

Run locally with `npm run dev`

Served from GitHub Pages at [fjordaan.github.io/fc-trails/trails/tree-trail](https://fjordaan.github.io/fc-trails/trails/tree-trail/)

The app is available at:  
- [trails.fulhamcemeteryfriends.org.uk](trails.fulhamcemeteryfriends.org.uk) - app home screen, listing all trails, with trails at:
- [trails.fulhamcemeteryfriends.org.uk/tree-trail](trails.fulhamcemeteryfriends.org.uk/tree-trail)  
- [trails.fulhamcemeteryfriends.org.uk/grave-trail](trails.fulhamcemeteryfriends.org.uk/grave-trail)  
- [trails.fulhamcemeteryfriends.org.uk/admin](trails.fulhamcemeteryfriends.org.uk/admin) - the CMS, requiring a GitHub PAT
